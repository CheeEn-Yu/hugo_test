工程數學-線性代數

教師: 李宏毅 教授
課程網址: 連結
心得: 六次python程式作業都是簡單又有收穫

1. cycle detection

利用簡單的矩陣乘法和一些處理，來解決圖論問題中，一個graph是否有環的問題

2. Hill Cipher

了解到乘一個矩陣，就像是將一個信息過一個線性系統，例如作業二就是一個加密系統

3. Cosine transform and its application

個人認為最好玩的一份作業，裡面有三題:
(a) 將聲音訊號轉到頻率域，濾掉雜音->1維DCT
(b) jpeg壓縮->二維DCT
(c) (bonus)助教加密圖片，透過逆操作解出原圖
這份作業實作了線性代數裡面change coordinate的觀念，先刻出一個basis，也另外隱隱指出其實那些cos也是vector
有趣的是可以自己選擇圖片壓縮，就可以看到壓縮完以後size變原本的0.2倍

4. PageRank

google 最一開始的演算法，被稱為最有價值的一個eigenvector

5. Linear regression

這個就是用上課丁丁公式直接算出最佳解，機器學習會有其他方法可以train出答案

6. SVD for Image compression

SVD:Singular value decomposition，看名字就知道是將任一矩陣拆開， 變成三個矩陣相乘，中間矩陣a_ij當i!=j時等於0(類似對角矩陣)。 可簡單理解成對角化的進階版，其中對角的值就叫做singular value。numpy一行code就能找出這三個矩陣了。
